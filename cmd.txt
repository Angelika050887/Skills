Cmd.exe — интерпретатор командной строки для операционных систем OS/2, Windows CE и для семейства операционных систем, базирующихся на Windows NT. 

Основные команды для работы с файловой системой:
1) CD – смена директории 
<диск>: или CD /d C:\- смена диска
C:\ - корень диска
cd test1 -перейти в папку test1

2) SYSTEMINFO - сбор информации о системе (ОС, имя компьютера, разрядность процессора и т.д.)
3) MD – создание директории
md test1- создать папку test1

4) RD – удаление директории
Удаление папки, если она не пустая: rd /s test1
Удаление папки, если она пустая: rd test1

5) SHUTDOWN используется для завершения сеанса пользователя, перезагрузки компьютера, перевода его 
в спящий режим или выключения питания.
/s - Завершение работы компьютера.
/r - Завершение работы и перезагрузка компьютера.
/a - Отмена завершения работы системы. Этот параметр можно использовать только в период ожидания.

6) MOVE — переместить из одной папки в другую один или несколько файлов

7) COPY — копировать в другое место один или несколько файлов

8) DEL / ERASE — удалить один или несколько файлов

9) REN / RENAME — переименовать файл или папку (какой-в какой/ какая- в какую)
rename 1.txt 2.txt 

10) REPLACE — заместить файлы

11) DIR- отображает все папки и файлы на диске

Другие команды (работа с сетью и не только):
1) Узнать, установлена ли java на компьютере и её версию
java -version

2) ping – проверяет доступность сетевого узла

3) ipconfig – выводит информацию о сетевых подключениях

4) tracert – определение маршрутов следования данных в сетях TCP/IP. 
Трассировка маршрута может показать скорость прохождения пакетов между маршрутизаторами, 
которые соединяют ПК, с которого идут запросы, и конечный сервер.
Трассировка маршрута помогает определить проблемный узел.

5) getmac (ipconfig /all)– узнать MAC-адреса сетевых устройств на своём компьютере
