Linux terminal commands (работа с файлами):

1) PWD (print working directory) – команда для вывода в терминал пути к текущей папке.
Синтаксис - $ pwd опции
Полезные опции:
--help - отобразить справку по утилите.

2) ls -просмотреть содержимое текущей папки (навигаия по файловой системе).
Синтаксис - $ ls опции /путь/к/папке
Команда ls -la отображает все  файлы, в том числе и скрытые, с указанием подробной информацией о них.

2) CAT – команда для вывода содержимого файла с начала.
TAC – команда для вывода содержимого файла с конца
Синтаксис - $ cat опции файл1 файл2 ...
Полезные опции:
-n - нумеровать все строки;
-s - удалять пустые повторяющиеся строки.

3) При помощи команды cat можно создать файл и можно написать текст сразу в этот файл:
cat > filename.txt 
Для сохранеения и выхода нужно нажать Ctrl+D.

4) CD (change directory) – команда для смены текущего (рабочего) каталога.
Синтаксис - $ cd опции папка_назначения
Полезные опции:
-e - если папку, в которую нужно перейти не удалось найти - выдает ошибку;
.. возвращает в корневую папку;
- показывает полный путь к директории.

5) MKDIR (make directory) – команда для создания новых директорий.
Синтаксис - $ mkdir опции директория
Полезные опции:
-p – создать все директории, указанные внутри пути (вложенные друг в друга директории).
Пример:
5.1) mkdir -p terminal/lnx/hey
создание 3 папок, вложенных в друг друга.
5.2) mkdir g1 g2 g3
создание одновременно 3-х папок, не вложенных друг в друга

6) CP (copy) – команда для копирования файлов и директорий
Синтаксис - $ cp опции файл-источник файл-приёмник 
Полезные опции:
-n - не перезаписывать существующие файлы;
-r - копировать папку Linux рекурсивно;
-u - скопировать файл, только если он был изменён;
Можно создавать новые файлы, указывая файл-приёмник, которого не существует.

7) MV (move) – команда для перемещения/переименования файлов и директорий.
Синтаксис - $ mv опции исходный_файл куда перемещаем 
Полезные опции:
-f – перезапись без разрешения пользователя;
-n – не перезаписывать существующие файлы.

8) TOUCH – команда для создания нового пустого файла.
Синтаксис - $ touch опции имя_файла
Полезные опции:
-m – устанавливает время последнего изменения файла.
Пример:
8.1) touch r1.txt r2.txt r3.txt j1.json j2.json
создание 5 новых файлов одновременно.

9) RM (remove) – команда для удаления файлов и директорий
Синтаксис - $ rm опции файл
Полезные опции:
-r – удаление директорий и их содержимого;
-d – удаление пустых директорий.

10) FIND – команда для поиска файлов и каталогов на основе специальных условий
Синтаксис - $ find [где_искать] [параметры] критерий шаблон
Полезные критерии:
-name – поиск файлов по имени;
-iname - игнорируется регистр текста (заглавные и строчные буквы понимаются одинаково);
-group – поиск по группе;
-user – поиск файлов по владельцу;
[параметры] по максимальной глубине поиска: 1-поиск в текущем каталоге, 2- в текущем каталоге и в папке выше
шаблон пишем в кавычках (дата, формат, имя и т.д.)
Примеры:
10.1) find . -name "abc*" 
поиск файлов с именем, начинающихся на abc;
10.2) find . -name "*.txt"
поиск всех файлов формата txt;
10.3) find . -not -name "abc*"
поиск всех файлов, начинающихся НЕ с abc;
10.4) find ./Desktop/ ./Documents/ -name "*.jpg"
поиск в нескольких директориях (папках).

11) GREP – команда для поиска строк, содержащих заданный пользователем образец.
Синтаксис - $ grep опции шаблон имя_файла
Полезные опции:
-r – рекурсивный поиск по каталогу;
-i – регистронезависимый поиск.
Примеры:
11.1) grep -A5 summary abc.txt
-A5 (after) выводит дополнительно 5 строк после строки, содержащей шаблон summary;
11.2) grep -B5 summary abc.txt
-B5 (before) выводит 5 строк перед строкой, содержащей шаблон summary;
11.3) grep -C5 summary abc.txt
-C5 выводит по 5 строк до и после искомой строки с шаблоном summary;
11.4) grep 'first' words.txt
вывести из файла «words.txt» все строки, 
содержащие слово «first».

12) SORT – команда для вывода строк, в определенном порядке (сортировка).
Синтаксис - $ sort опции файл
Полезные опции:
-b - не учитывать пробелы;
-d - использовать для сортировки только буквы и цифры;
-r - сортировать в обратном порядке.
Примеры:
12.1) sort debts.txt – сортировка по алфавиту;
12.2) sort -r debts.txt    – сортировка в обратном порядке.

13) DIFF сравнивает файлы
Синтаксис diff [опции] файл1 файл2
Полезные опии: 
-d - выводит только отличия;
-s - выводит только совпадающие части;
-y - вывод в формате 2-х колонок;
-r - рекурсивное сравнение каталогов;
-i - игнорировать регистр.

14) OPEN - открыть файл
Синтаксис open [опции] файл
